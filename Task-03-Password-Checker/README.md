# Task-03: Password Strength Checker Tool

## 🔐 Overview
This is a simple Python-based tool that checks the strength of a password based on security rules.

## ⚙️ Features
- Checks password length
- Detects uppercase and lowercase letters
- Checks numbers
- Checks special characters
- Gives strength level (Weak / Medium / Strong)

## ▶️ How to Run

```bash
python3 password_tool.py
