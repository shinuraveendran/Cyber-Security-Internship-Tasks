# Task-05: Network Packet Analyzer

## ⚠️ Disclaimer
This project is created for educational purposes only and should be used with proper authorization.

## 🔐 Overview
This tool captures and analyzes live network packets.

## ⚙️ Features
- Captures network traffic in real-time
- Displays source and destination IP addresses
- Identifies protocols (TCP/UDP)
- Shows packet payload data

## 🛠️ Technologies Used
- Python
- Scapy

## ▶️ How to Run

```bash
sudo python3 task5_packet_sniffer.py
