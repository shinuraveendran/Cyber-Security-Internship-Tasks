# Task-01: Caesar Cipher Tool

## Description
This project implements a simple Caesar Cipher algorithm in Python for encrypting and decrypting text using a shift key.

## Features
- Encrypt text
- Decrypt text
- User input support

## How to Run
1. Run the Python file
2. Enter message
3. Enter shift value
4. Choose encrypt/decrypt

## Technologies Used
- Python
