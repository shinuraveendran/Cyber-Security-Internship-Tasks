# Task-02: Image Encryption Tool

## 🔐 Overview
This project is a simple image encryption and decryption tool using pixel manipulation.

## ⚙️ Features
- Encrypt image using a key
- Decrypt image back to original
- Simple CLI-based tool

## 🛠️ Technologies Used
- Python
- Pillow (PIL)

## ▶️ How to Run

```bash
python3 task2_image_tool.py
